    Commit! You deserve a good commit tool.

    Using Git, you are about to build a code history. The Git Changes Window inside Visual Studio detects the file changes and dispalys the diff in real time. You can review the changes and select files to create meaningful commits.

    Git History A different view.

    Git history viewer is an unique tool that displays the commit graph horizontally. It provides simplified view mode to only show important commits to help you understand the history structure. It also can be used to view the details of each commit, blame file, create/delete branches and tag, rebase, cherry-pick, stash and etc.
