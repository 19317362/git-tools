    Git tools for Visual Studio

    • Install Git for Windows, Git Extensions or TortoiseGit.
    • Run Visual Studio.
    • Select Menu Tools | Extensions and Updates..., search online gallery for Git Tools and install.
    • Select Menu 'Git Tools' or add 'Git Tools' toolbar to the main window.
