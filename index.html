    Commit! You deserve a good tool.

    Using Git, you commit a lot. The Git Changes Window inside Visual Studio detects the file changes and dispalys the diff in real time. You can review the changes and select files to create meaningful history.

    Git History! You can have a different view.

    Git history viewer is an unique tool that displays the commits horizontally to help you understand the history. Its simplified view mode highlights the history structure by only showing important commits . It also can be used to view the details of each commit, blame file, create/delete branches and tag, rebase, cherry-pick, stash and etc.

    Differences! You can edit files in diff mode.

    From the Git Changes Window, you can launch the Visual Studio Diff Viewer with the comitted version of files and the version being edited. It provides a clear view to see the changes/differences between versions.

    Note: Git for Windows is required in order to for the tools above. Git Extensions or TortoiseGit are optional.
